# Build a Music Shopping Cart Page

**Description:**  
In this lab, I built a responsive music shopping cart page using semantic HTML and Tailwind CSS. The page displays a collection of music equipment products, each with quantity controls, pricing information, and remove buttons, alongside an order summary containing the subtotal, shipping information, total cost, and checkout button.

The goal of this exercise was to practice building responsive layouts with Tailwind CSS utility classes, organizing content using Flexbox, applying consistent spacing and typography, and creating reusable card-style components for an e-commerce interface.

The project also reinforced the use of semantic HTML elements, responsive design principles, and utility-first CSS workflows for creating clean, maintainable user interfaces.

**Technologies Used:**

- HTML5
- Tailwind CSS
- Responsive Design
- Flexbox
- Semantic HTML

**File Structure:**

```text
build-a-music-shopping-cart-page/
├── index.html   # Shopping cart page structure and Tailwind styling
```

**Setup / How to Run:**

1. Clone or download the repository.
2. Open the project folder.
3. Open `index.html` in your browser.
4. View the responsive shopping cart layout.
5. Resize the browser window to observe the responsive design.
