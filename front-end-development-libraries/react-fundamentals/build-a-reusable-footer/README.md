# Build a Reusable Footer Component

**Description:**  
In this lab, I built a reusable footer component using React. The footer is structured as a standalone component that can be rendered anywhere within an application. It contains multiple navigation sections, a copyright message, and a row of social/media icons.

The goal of this exercise was to practice creating a functional React component and rendering it to the DOM using ReactDOM. This project demonstrates how JSX can be used to define UI structure while keeping the component modular and reusable.

**Technologies Used:**

- HTML
- CSS
- JavaScript (ES6)
- React (Functional Components)
- JSX
- ReactDOM

**File Structure:**

```
build-a-reusable-footer/
├── index.html   # Root HTML file that loads React and renders the component
├── styles.css   # Styling for the footer layout and visual design
├── index.jsx    # React Footer component
└── README.md    # Project documentation
```

**Setup / How to Run:**

1. Clone or download the repository.
2. Open the project folder.
3. Open `index.html` in your browser.
4. The React application will render the reusable Footer component inside the root element.

**Learning Outcomes:**

- Creating a React functional component
- Writing UI using JSX syntax
- Rendering React components to the DOM using ReactDOM
- Structuring reusable UI components
- Applying CSS styling to React components
- Understanding how HTML, CSS, and React work together to build modular interfaces
