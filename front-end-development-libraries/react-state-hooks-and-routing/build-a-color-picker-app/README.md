# Build a Color Picker App

**Description:**  
In this lab, I built an interactive color picker application using React functional components and state management. The application allows the user to select a color using an input element, and dynamically updates the background color of the page based on the selected value.

The goal of this exercise was to practice working with React state using the `useState` hook, handling user input events, and creating dynamic UI updates based on state changes.

**Technologies Used:**

- HTML
- CSS
- JavaScript (ES6)
- React (Functional Components)
- JSX
- ReactDOM

**File Structure:**

```
build-a-color-picker-app/
├── index.html   # Root HTML file that loads React and renders the ColorPicker component
├── styles.css   # Styling for layout and visual presentation
├── index.jsx    # React ColorPicker component with state and event handling
└── README.md    # Project documentation
```

**Setup / How to Run:**

1. Clone or download the repository.
2. Open the project folder.
3. Open `index.html` in your browser.
4. Use the color input to dynamically change the background color.

**Learning Outcomes:**

- Managing component state using the `useState` hook
- Handling form input and user events in React
- Creating dynamic UI updates based on state changes
- Using inline styles in JSX
- Structuring simple interactive components
- Rendering React components to the DOM using ReactDOM
