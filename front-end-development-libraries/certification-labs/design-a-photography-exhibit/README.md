# Design a Photography Exhibit

**Description:**  
In this lab, I built a responsive photography exhibit landing page using Tailwind CSS utility classes. The application displays a collection of photography cards in a responsive grid layout that automatically adjusts from a single-column layout on smaller screens to a three-column layout on larger devices.

The goal of this exercise was to practice styling user interfaces with Tailwind CSS, applying responsive design principles, and building reusable card layouts using utility-first classes. The project also demonstrates the use of Tailwind’s spacing, typography, colors, borders, hover states, and Flexbox and Grid utilities to create a clean, responsive design without writing custom CSS.

**Technologies Used:**

- HTML5
- Tailwind CSS
- Responsive Design
- CSS Grid
- Flexbox

**File Structure:**

```
design-a-photography-exhibit/
├── index.html
└── README.md
```

**Setup / How to Run:**

1. Clone or download the repository.
2. Open the project folder.
3. Open `index.html` in your browser.
4. View the layout.
5. Resize the browser window to observe the responsive design.

**Learning Outcomes:**

- How to rapidly build user interfaces using Tailwind CSS utility classes
- Applying responsive breakpoints with Tailwind (md:)
- Structuring layouts using CSS Grid and Flexbox utilities
- Creating reusable card components with consistent styling
- Implementing hover states and button styling without custom CSS
- Using Tailwind’s spacing, typography, border, and color utility classes to produce clean, maintainable layouts
