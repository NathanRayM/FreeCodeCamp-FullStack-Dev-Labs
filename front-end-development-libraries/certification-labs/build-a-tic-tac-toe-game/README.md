# Build a Tic-Tac-Toe Game

**Description:**  
In this lab, I built an interactive Tic-Tac-Toe game using React functional components and hooks. The application allows two players to take turns placing X and O marks on a 3x3 game board while dynamically tracking the current game state.

The goal of this exercise was to practice managing multiple pieces of state in React, handling user interaction through event handlers, updating arrays immutably, and implementing conditional game logic such as detecting winners and draw conditions.

The app also demonstrates how to prevent invalid moves, stop gameplay after the game has ended, dynamically update status messages, and reset the game state using React state management patterns.

**Technologies Used:**

- HTML
- CSS
- JavaScript (ES6)
- React (Functional Components)
- JSX
- React Hooks (useState)
- ReactDOM

**File Structure:**

```
build-a-tic-tac-toe-game/
├── index.html   # Root HTML file that loads React and renders the Board component
├── styles.css   # Styling for layout and visual presentation
├── index.jsx    # React Tic-Tac-Toe game logic and components
└── README.md    # Project documentation
```

**Setup / How to Run:**

1. Clone or download the repository.
2. Open the project folder.
3. Open index.html in your browser.
4. Click the squares to play the game.
5. Use the reset button to start a new game.
