# Build a Mood Board

**Description:**  
In this lab, I built a dynamic destination mood board using React functional components. The application renders multiple mood board items by mapping over an array of destination data and passing props to reusable components. Each item displays an image, description, and background color to represent different travel locations.

The goal of this exercise was to practice creating reusable React components, passing props, dynamically rendering lists using `.map()`, and integrating styling with JSX.

**Technologies Used:**

- HTML
- CSS
- JavaScript (ES6)
- React (Functional Components)
- JSX
- ReactDOM

**File Structure:**

```
build-a-mood-board/
├── index.html   # Root HTML file that loads React, and renders the MoodBoard component
├── styles.css   # Styling for the mood board layout and visual design
├── index.jsx    # React MoodBoard and MoodBoardItem components
└── README.md    # Project documentation
```

**Setup / How to Run:**

1. Clone or download the repository.
2. Open the project folder.
3. Open `index.html` in your browser.
4. The React application will render the Mood Board inside the root element.

**Learning Outcomes:**

- Creating reusable React functional components
- Passing and destructuring props in React
- Dynamically rendering UI using `.map()`
- Structuring component-based UI architecture
- Integrating CSS styling with JSX
- Rendering React components to the DOM using ReactDOM
