# Debug a random background color changer

**Description:**  
In this lab, you'll debug the javaScript of a random background color changer.

**Technologies Used:**

- HTML
- CSS
- JavaScript

**File Structure:**

```
build-a-set-of-football-team-cards/
├── index.html   # Main HTML file with structure
├── style.css    # Styles for the page layout
├── script.js    # JavaScript for selecting random bakground colors
└── README.md    # This file
```

**Setup / How to Run:**

1. Clone or download the repository.
2. Navigate to `JavaScript/debugging/build-a-set-of-football-team-cards/`.
3. Open `index.html` in a web browser.
4. Click the button to change to different background colors.

**Learning Outcomes:**

- Handling click events with `addEventListener`.
- Using functions, Math.random, Math.floor to generate a random background color.
- Learning debugging concepts and best practices.
