# Debug a Random Background Color Changer

**Description:**  
In this lab, you will debug a random background color changer.

**Technologies Used:**

- HTML
- CSS
- JavaScript

**File Structure:**

```
debug-a-random-background-color-changer/
├── index.html   # Main HTML file with structure
├── style.css    # Styles for the page layout
├── script.js    # JavaScript for changing colors of the background.
└── README.md    # This file
```

**Setup / How to Run:**

1. Clone or download the repository.
2. Navigate to `JavaScript/debugging/debug-a-random-background-color-changer/`.
3. Open `index.html` in a web browser.
4. Change the background by clicking the button.

**Learning Outcomes:**

- Handling click events with `addEventListener`.
- Using functions, Math.floor, and Math.random to select a random color
- learning the process of debugging javaScript
