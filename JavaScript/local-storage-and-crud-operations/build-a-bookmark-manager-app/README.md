# Build a Bookmark Manager App

**Description:**  
This project is a vanilla JavaScript Bookmark Manager application built as part of FreeCodeCamp’s JavaScript curriculum. The app allows users to create, view, filter, and delete bookmarks, with all data persisted using the browser’s localStorage.

**Technologies Used:**

- HTML
- CSS
- JavaScript

**File Structure:**

```
build-a-bookmark-manager-app/
├── index.html   # Main HTML file with structure
├── style.css    # Styles for the page layout
├── script.js    # JavaScript for functionality
└── README.md    # This file
```

**Setup / How to Run:**

1. Clone or download the repository.
2. Navigate to `JavaScript/local-storage-and-crud-operations/build-a-bookmark-manager-app/`.
3. Open `index.html` in a web browser.
4. Enter markdown input get the raw HTML and an HTML preview

**Learning Outcomes:**

- Working with localStorage (get, set, parse, stringify)
- Defensive data validation and normalization
- DOM manipulation and event handling
- Filtering and updating arrays immutably
- Managing application state without frameworks
- Building reusable helper functions
- Handling edge cases and user interactions
