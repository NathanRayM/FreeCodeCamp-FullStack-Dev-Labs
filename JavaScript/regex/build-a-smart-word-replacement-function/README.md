# Build a Smart Word Replacement Function

**Description:**  
In this lab, you will create a function that performs a search and replace on the sentence using the arguments provided and then returns the new sentence.

**Technologies Used:**

- JavaScript

**File Structure:**

```
build-a-smart-word-replacement-function/

└── README.md    # This file
```

1. Clone or download the repository.
2. Open the project in your code editor.
3. Run the script in your browser console or Node.js:

**Learning Outcomes:**

- Understanding how to use regular expressions to search for dynamic patterns in a string.
- Using the RegExp constructor to build patterns from variables.
- Applying callback functions inside String.prototype.replace() to customize replacement logic.
- Detecting and preserving capitalization when transforming words.
- Working with string methods such as .charAt(), .slice(), and .toUpperCase().
- Practicing conditional logic using the ternary operator or traditional if/else statements.
- Gaining experience with dynamic text manipulation and string processing in JavaScript.
