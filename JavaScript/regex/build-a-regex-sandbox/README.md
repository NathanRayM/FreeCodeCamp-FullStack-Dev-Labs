# Build a Regex Sandbox

**Description:**  
By completing the Regex Sandbox project, you will strengthen your understanding of JavaScript, regular expressions, and interactive web interfaces.

**Technologies Used:**

- HTML
- CSS
- JavaScript

**File Structure:**

```
build-a-regex-sandbox/
├── index.html   # Main HTML file with structure
├── style.css    # Styles for the page layout
├── script.js    # JavaScript for functionality to check if input matches the regex pattern
└── README.md    # This file
```

**Setup / How to Run:**

1. Clone or download the repository.
2. Navigate to `JavaScript/regex/build-a-regex-sandbox/`.
3. Open `index.html` in a web browser.
4. Input a word and click check to see if it's a palindrome.

**Learning Outcomes:**

- Using regular expressions to test, match, and highlight parts of a string
- Applying regex flags to control search behavior
- Building dynamic RegExp objects from user input
- DOM manipulation using querySelector, innerHTML, and event listeners
- Updating the page based on user interactions
- Working with conditional logic to determine match results
