# Build a theme switcher

**Description:**  
In this lab, you will build an app that switches between different themes. When you switch to a different theme, the background color will change and a different message will display.

**Technologies Used:**

- HTML
- CSS
- JavaScript

**File Structure:**

```
build-a-theme-switcher/
├── index.html   # Main HTML file with structure
├── style.css    # Styles for the page layout
├── script.js    # JavaScript for selecting themes listed in the dropdown menu
└── README.md    # This file
```

**Setup / How to Run:**

1. Clone or download the repository.
2. Navigate to `JavaScript/javascript-and-accessibility/build-a-theme-switcher/`.
3. Open `index.html` in a web browser.
4. Select team positions from the dropdown.

**Features / Functionality:**

- Select a theme from the dropdown button
- Selections show different themes

**Learning Outcomes:**

- Handling click events with `addEventListener`.
- Using functions and forEach to help filter through the object.
- Reinforcing the concept of DOM manipulation and interactivity with JavaScript.
- creating a dropdown that allows for other selections within the menu
