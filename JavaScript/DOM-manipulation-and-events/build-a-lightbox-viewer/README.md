# Build a Lightbox Viewer

**Description:**  
In this lab, you'll build a lightbox viewer for viewing images in a focused mode.
You'll practice click events and toggling classes.

**Technologies Used:**

- HTML
- CSS
- JavaScript

**File Structure:**

```
build-a-lightbox-viewer/
├── index.html   # Main HTML file to showcase photos
├── style.css    # Styles and layout for the page
├── script.js    # JavaScript for selecting photos from the lightbox viewer
└── README.md    # This file
```

**Setup / How to Run:**

1. Clone or download the repository.
2. Navigate to `JavaScript/DOM-manipulation-and-events/build-a-lightbox-viewer/`.
3. Open `index.html` in a web browser.
4. Click on the heart icons to toggle them between transparent and red.

**Features / Functionality:**

- Click on favorite icons to toggle their state.
- Visual feedback shows which items are marked as favorite.
- Smooth toggle using CSS styling changes.

**Learning Outcomes:**

- Handling click events with `addEventListener`.
- Selecting multiple DOM elements using `querySelectorAll`.
- Using forEach to iterate through numerous classes.
- Reinforcing the concept of DOM manipulation and interactivity with JavaScript.
