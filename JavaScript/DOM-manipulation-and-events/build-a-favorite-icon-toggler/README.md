# Build a Favorite Icon Toggler

**Description:**  
This lab demonstrates how to use JavaScript to toggle elements in the DOM. Users can click on a favorite icon (heart) to toggle its state between transparent and red, providing a visual cue for favorites.

**Technologies Used:**

- HTML
- CSS
- JavaScript

**File Structure:**

```
build-a-favorite-icon-toggler/
├── index.html   # Main HTML file with favorite icons
├── style.css    # Styles for the icons and page layout
├── script.js    # JavaScript for toggling icons on click
└── README.md    # This file
```

**Setup / How to Run:**

1. Clone or download the repository.
2. Navigate to `JavaScript/DOM-manipulation-and-events/build-a-favorite-icon-toggler/`.
3. Open `index.html` in a web browser.
4. Click on the heart icons to toggle them between transparent and red.

**Features / Functionality:**

- Click on favorite icons to toggle their state.
- Visual feedback shows which items are marked as favorite.
- Smooth toggle using CSS styling changes.

**Learning Outcomes:**

- Handling click events with `addEventListener`.
- Toggling CSS classes to update element styles.
- Selecting multiple DOM elements using `querySelectorAll`.
- Reinforcing the concept of DOM manipulation and interactivity with JavaScript.
