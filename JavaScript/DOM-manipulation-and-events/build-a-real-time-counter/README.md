# Build a real time counter

**Description:**  
This lab demonstrates how to work with the input event when the user types in the input field.

**Technologies Used:**

- HTML
- CSS
- JavaScript

**File Structure:**

```
build-a-favorite-icon-toggler/
├── index.html   # Main HTML file with favorite icons
├── style.css    # Styles for the icons and page layout
├── script.js    # JavaScript for toggling icons on click
└── README.md    # This file
```

**Setup / How to Run:**

1. Clone or download the repository.
2. Navigate to `JavaScript/DOM-manipulation-and-events/build-a-real-time-counter/`.
3. Open `index.html` in a web browser.
4. Type in the text box to see the counter change.

**Features / Functionality:**

- Type in the text box to affect the counter
- Visual feedback shows the counter change as the number of characters increase.

**Learning Outcomes:**

- Handling input events with `addEventListener`.
- Using CSS classes to update element styles.
- Reinforcing the concept of DOM manipulation and interactivity with JavaScript.
