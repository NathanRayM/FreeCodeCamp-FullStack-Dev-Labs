# Build a set of football team cards

**Description:**  
In this lab, you'll use DOM manipulation, object destructuring, event handling, and data filtering to build a set of football team cards.

**Technologies Used:**

- HTML
- CSS
- JavaScript

**File Structure:**

```
build-a-set-of-football-team-cards/
├── index.html   # Main HTML file with structure
├── style.css    # Styles for the cards and page layout
├── script.js    # JavaScript for selecting positions listed in the dropdown menu
└── README.md    # This file
```

**Setup / How to Run:**

1. Clone or download the repository.
2. Navigate to `JavaScript/DOM-manipulation-and-events/build-a-set-of-football-team-cards/`.
3. Open `index.html` in a web browser.
4. Select team positions from the dropdown.

**Features / Functionality:**

- Select team position from the dropdown to view those players
- Cards show specific palyer positions

**Learning Outcomes:**

- Handling click events with `addEventListener`.
- Using functions and forEach to help filter through the object.
- Reinforcing the concept of DOM manipulation and interactivity with JavaScript.
